# ToDo-bot
The best bot for your productivity :)

## Запуск web-сервера
### Командная строка
`run.bat`
### PowerShell
`.\run.ps1`
### linux
Выполните в консоли команду `chmod +x run.sh` - это сделает файл исполняемым.
Теперь для запуска проекта нужно писать `./run.sh`

## Запуск бота
`python bot_app`
